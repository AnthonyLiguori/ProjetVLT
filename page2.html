Lists and tables

Lists :

  • Ordered list (ol) : This is a numbered list where each item in the list is preceded by a number or letter.
  • Unordered list (ul) : This is a bulleted list where each item in the list is preceded by a period.
  • TDefinition list (dl) : This is a list that defines terms and their descriptions. Each term is associated with a description, which is usually a paragraph element.
